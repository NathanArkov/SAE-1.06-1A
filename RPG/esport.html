Implication des RPG dans l'esport

Le Speedrun

Le speedrun est une discipline consistant à finir un jeu le plus rapidement possible.
Il existe énormément de catégorie de speedrun. Nous avons par exemple speedrun sans glitch ou avec certains glitch.
L’eSport étant la partie compétitive des jeux, le speedrun compare les temps des "run", le gagnant étant celui qui effectue le run le plus court.
Par rapport à d'autres compétitions, le speedrun n'est pas un moyen de gagner sa vie. Il existe toutefois le tournois "le Barrière SpeedRun Show", une compétition française créée en 2020 avec un cashprize de 10 000€.
Donc les RPG ne sont pas trop compétitifs, par rapport à d'autre style de jeux

cashprize = prix pour les vainqueurs de la compétition.
Glitch = est la traduction de "problème". C'est une fonctionnalité que le créateur du jeu n'a pas prévu, Le glitch permet d'avancer beaucoup plus vite dans certains jeux.
Run = c'est une partie d’un jeu que l'on chronomètre, le meilleur run est votre meilleur temps pour finir le jeux.
Pour plus d'informations sur les speedrun ou sur les catégories de speedrun sur le site : Speedrun.com
